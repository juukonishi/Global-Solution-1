## O que o projeto faz

Sistema de monitoramento de missão espacial desenvolvido em Python. O programa simula ciclos operacionais de uma missão, analisa automaticamente dados de temperatura, comunicação, bateria, oxigênio e estabilidade, gera alertas por nível de risco (NORMAL / ATENÇÃO / CRÍTICO) e exibe um relatório final com tendência da missão, área mais afetada e recomendações automáticas.
